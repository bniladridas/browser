Settings Guide

Everything that powers the browser on the desktop lives in the floating settings panel. Toggle once and the behavior applies instantly without a restart.

Quick toggles

  • Hide App Bar removes the address and toolbar for a more immersive view.
  • Legacy User Agent swaps the default string to a legacy browser when a site misbehaves.
  • Git Fetch enables the GitHub repo picker in the Torry feed.
  • Private Browsing keeps history and cookies out of persistent storage.
  • Ad Blocking applies the built-in blocklist to every navigation.
  • Strict Mode disables JavaScript where necessary for highly sensitive pages.
  • Ambient Mode implements a dynamic, glowing background that adapts to the page content.

Password Manager

Flip this toggle to turn on the JavaScript detector that observes traditional login forms and surfaces the save prompt.

  • The prompt only appears when the login form is a standard username/email + password submission.
  • Sites that automatically switch to passkeys, multi-step iframes, or progressive flows (e.g., Google sign-in when you already have a passkey) won’t fire the detector.
  • You can still manually add entries through the Password Vault screen that shows when the toggle is active.
  • Use the “Never save for this site” preference (stored silently in your settings) to skip prompts for shared machines.

Media & security

Camera and microphone access is granted automatically as soon as a page asks for them, so services like Meet, Zoom, or verification forms stop hanging on “plugged-in” warnings.

  • The browser now calls the permission handler built into webview_flutter and accepts getUserMedia requests for audio/video resources.
  • macOS will still prompt once with the OS dialog because the app declares NSCameraUsageDescription and NSMicrophoneUsageDescription.
  • If a site tries to capture other platform resources, the request is denied instantly so the page fails fast.

Advanced helpers

  1. Reorderable Tabs lets you drag tab buttons on the strip to keep frequently used ones close.
  2. AI Search Suggestions surfaces the TinyAI-compiled prompts when you type in the address bar.
  3. Advanced Cache keeps some resources around longer for faster page reloads when you revisit a site.

You can change the global theme from the modal and it updates every view instantaneously, including the sidebar and blog pages.
